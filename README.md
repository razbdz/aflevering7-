how to
Åbn projektet i Rider eller Visual Studio.

Tryk Run (grøn trekant).

Klik på én af knapperne i toppen:

Process Next Order eller Process Order (Robot)

Se opdateringer i kø, lager og status.




_______


To knapper øverst i vinduet:

Process Next Order → Behandler næste ordre manuelt (uden robot).

Process Order (Robot) → Behandler næste ordre og viser robotaktivitet trin for trin.

Queued Orders viser de ordrer, der venter på behandling.

Processed Orders viser færdigbehandlede ordrer.

Total Revenue viser den samlede omsætning.

Inventory viser hvor mange varer, der er tilbage på lageret.

Statusfelt nederst viser alle beskeder, fx robotbevægelser og status på ordrer.
